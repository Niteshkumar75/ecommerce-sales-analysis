# E-commerce Sales Data Analysis

## 📊 Project Overview
This project analyzes e-commerce sales data using Python.

## 🔧 Tasks Performed
- Data cleaning (missing values, duplicates, inconsistent data)
- Data transformation (date conversion, formatting)
- Data analysis (total revenue, city-wise revenue)
- Data visualization using bar charts

## 📈 Key Insights
- Berlin generated the highest revenue
- Cleaned data improved accuracy of analysis

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
