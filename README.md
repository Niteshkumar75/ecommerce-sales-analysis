# 🛒 E-commerce Sales Data Analysis

## 📌 Project Overview
This project focuses on cleaning and analyzing raw e-commerce sales data using Python. The dataset contained missing values, inconsistent formatting, and duplicate records, which were cleaned before analysis.

---

## 🎯 Objectives
- Clean raw data for accurate analysis  
- Analyze total revenue and city-wise sales  
- Identify top-performing locations  
- Visualize insights using charts  

---

## 🧹 Data Cleaning Steps
- Filled missing values using mean  
- Removed rows with missing customer data  
- Standardized city names (e.g., Berlin, berlin → BERLIN)  
- Removed duplicate records  
- Converted date column to proper datetime format  

---

## 📊 Analysis Performed
- Total revenue calculation  
- Revenue grouped by city  
- Time-based sales analysis  

---

## 📈 Key Insights
- Berlin generated the highest revenue  
- Data cleaning significantly improved analysis accuracy  
- Visualization helped identify business trends  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

---

## 🚀 Conclusion
This project demonstrates how raw data can be transformed into meaningful insights using data cleaning and analysis techniques.
